📅 Meeting Coordinator

A meeting scheduling assistant that automates the process of creating professional meeting events and integrating them with Google Calendar. The system uses Google's Gemini AI model to generate refined meeting titles and automatically schedules meetings with attendees.
AI Meeting Coordinator is designed to simplify meeting scheduling by combining Generative AI with Google Calendar Automation.

The application allows users to:

Enter meeting details
Generate a professional meeting title using AI
Automatically create a Google Calendar event
Add attendees and send invitations
Receive a calendar event link instantly

This project demonstrates the integration of LLMs (Large Language Models) with external APIs to automate real-world workflows.

###Features:
AI-Powered Meeting Title Generation
Uses Google Gemini AI model
Converts user-provided topics into professional meeting titles
Applies prompt engineering techniques for concise outputs
#Google Calendar Integration
Automatically creates calendar events
Sends invitations to attendees
Generates event links
#Attendee Management
Supports multiple attendee email addresses
Automatically adds participants to the calendar event
#User-Friendly Interface
Built with Streamlit
Simple and intuitive form-based workflow
#Error Handling
Handles API failures gracefully
Uses fallback mechanisms when AI responses are unavailable
